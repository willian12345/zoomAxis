# 时间轴&轨道编辑示例
![image](screenshot/demo2.png)
运行环境要求
-----------------------------------------------------------------------------------------------------------
node >= v16.13.1
pnpm >= 6.11.0
vite >= 3.1.0
typescript >= 4.6.4

安装
-----------------------------------------------------------------------------------------------------------
pnpm i 


## demo vue3.0

进入 apps/vue-app

```
pnpm dev
```

## demo react

进入 apps/react-app

```
pnpm dev
```
